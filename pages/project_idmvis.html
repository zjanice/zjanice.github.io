IDMVis

Temporal Event Sequence Visualization for Type 1 Diabetes Treatment Decision Support

About The Visualization

IDMVis is an open source interactive visualization tool for showing type 1 diabetes patient data. It is designed to help clinicians perform temporal inference tasks: specifically for recommending adjustments to patient insulin protocol, diet, and behavior. IDMVis is built with JavaScript and the d3.js library.

Publication

Yixuan Zhang, Kartik Chanana, and Cody Dunne, "IDMVis: Temporal Event Sequence Visualization forType 1 Diabetes Treatment Decision Support", IEEE Transactions on Visualization and Computer Graphics (Proc. Information Visualization 2017), vol. 25, no. 01, Jan. 2019. Preprint.
