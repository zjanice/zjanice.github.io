Projects About

IDMVis: Temporal Event Sequence Visualization

IDMVis is an open source interactive visualization tool for showing type 1 diabetes patient data. It is designed to help clinicians perform temporal inference tasks: specifically for recommending adjustments to patient insulin protocol, diet, and behavior.

Food Pairing Visualization

In this project, we compare and contrast the use of food ingredients in different cuisines. We develop Cuisine Ingredient Matrix to show the popularity of the food ingredient in different cuisines, and what ingredient goes well with. We also create Cuisine Similarity Matrix to demonstrate how similar between two cuisines.

Airbnb Data Visualization

What do we learn from Airbnb popular hosts? This project aims at understanding how to help hosts increase Airbnb income using a visualization tool.

Baby Care Log Redesign

In this project, we conducted user research, redesigned the information architecture and wireframes of Baby Care Log - an Android application that tracks and monitors key aspects of baby's growth and development.

HydrateMe

HydrateMe is designed as a public display with social networking features to motivate people to keep hydrated. This project uses human-centered process to conduct user research and user study.
